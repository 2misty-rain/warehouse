# 项目状态

请快速检查并汇报以下信息（只读，不做任何修改）：

## 代码状态
- git 分支和未提交变更
- 后端路由总数
- 前端编译状态

## 数据库状态
- inventory_db 表数量和 inventory 设备总数
- wechat_idc 表和数据概况
- 监控设备数（商机交付+商机试用）

## 服务状态
- 服务器是否在 8000 端口运行
- 前端是否在 3000 端口运行
- 定时任务调度器状态

## 上次测试结果
- 如果有保存的测试记录，显示上次测试通过率
