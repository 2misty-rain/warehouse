# 智能AI助手使用说明

## 🎯 功能特性

升级后的AI助手具备以下强大能力：

### 1. **真正的智能对话**
- 使用阿里云通义千问API，具备深度思考和理解能力
- 能够理解复杂的自然语言查询
- 支持多轮对话和上下文理解

### 2. **实时数据分析**
- 基于当前库存数据进行智能分析
- 自动识别库存问题和趋势
- 提供数据驱动的建议

### 3. **智能预警和建议**
- 主动发现逾期设备并提醒
- 分析设备借用趋势
- 提供库存优化建议

### 4. **专业库存管理咨询**
- 回答库存管理相关问题
- 提供最佳实践建议
- 帮助优化管理流程

## 📋 配置步骤

### 1. 获取API密钥

1. 访问 [阿里云DashScope控制台](https://dashscope.console.aliyun.com/)
2. 注册/登录阿里云账号
3. 创建API密钥

### 2. 配置API密钥

在 `backend/.env` 文件中设置你的API密钥：

```env
DASHSCOPE_API_KEY=sk-your-actual-api-key-here
```

**重要提示：**
- 不要将 `.env` 文件提交到版本控制系统
- 确保API密钥保密，不要泄露给他人

### 3. 安装依赖

确保已安装 `requests` 库：

```bash
pip install requests
```

### 4. 启动服务

```bash
cd backend
python main.py
```

## 🔧 技术细节

### API端点
- **URL**: `POST /api/ai/chat`
- **请求格式**: `{"user_input": "你的问题"}`
- **响应格式**: 
```json
{
  "success": true,
  "reply": "AI的智能回复",
  "data": {
    "type": "ai_response",
    "timestamp": "2024-01-01T12:00:00"
  }
}
```

### 系统上下文
AI助手会自动获取以下实时数据作为对话上下文：
- 总设备数、可用设备数、借出设备数
- 活跃借用记录数、逾期记录数
- 设备类型分布
- 最近的借用记录（最多5条）

### AI模型参数
- **模型**: qwen-plus（通义千问）
- **温度**: 0.7（平衡创造性和准确性）
- **最大token**: 1500
- **超时时间**: 30秒

## ⚠️ 注意事项

1. **API费用**: 使用通义千问API会产生费用，请查看阿里云定价
2. **网络要求**: 需要能够访问阿里云API服务器
3. **速率限制**: API有调用频率限制，请勿频繁调用
4. **错误处理**: 如果API不可用，系统会返回错误信息

## 📝 故障排除

### 问题1: "DASHSCOPE_API_KEY 未配置"
**解决**: 确保在 `backend/.env` 文件中正确设置了API密钥

### 问题2: "AI服务暂时不可用"
**解决**: 
- 检查网络连接
- 验证API密钥是否正确
- 确认阿里云账户余额充足

### 问题3: 响应速度慢
**解决**: 
- 这是正常现象，AI推理需要时间
- 可以调整timeout参数（默认60秒）

## 🚀 进阶使用

你可以通过修改 `smart_ai.py` 中的 `get_system_context` 函数来添加更多上下文信息，让AI更了解你的业务。

例如添加：
- 历史趋势数据
- 季节性分析
- 用户行为模式
- 自定义业务规则
