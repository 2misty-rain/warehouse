# 🚀 快速配置智能AI助手

## 第一步：获取API密钥

1. 访问 [阿里云DashScope控制台](https://dashscope.console.aliyun.com/)
2. 登录你的阿里云账号（如果没有，先注册）
3. 点击"API-KEY管理"
4. 创建一个新的API密钥
5. 复制你的API密钥（格式类似：`sk-xxxxxxxxxxxxxxxx`）

## 第二步：配置API密钥

打开 `backend/.env` 文件，将你的API密钥填入：

```env
DASHSCOPE_API_KEY=sk-你的实际API密钥
```

**示例：**
```env
DASHSCOPE_API_KEY=sk-1234567890abcdef1234567890abcdef
```

## 第三步：重启后端服务

1. 停止当前运行的后端服务（如果正在运行）
2. 重新启动：
   ```bash
   cd backend
   python main.py
   ```

## 第四步：测试AI助手

### 方法1：通过前端界面
1. 打开浏览器访问前端应用
2. 找到"AI智能助手"组件
3. 尝试发送消息，例如：
   - "现在库存情况怎么样？"
   - "帮我分析一下当前的设备借用情况"
   - "有什么需要注意的问题吗？"

### 方法2：通过测试脚本
```bash
cd backend
python test_smart_ai.py
```

## ❌ 常见问题

### 问题1：提示"DASHSCOPE_API_KEY 未配置"
**解决**：检查 `.env` 文件是否正确保存，确保没有多余的空格

### 问题2：AI回复很慢或超时
**解决**：
- 这是正常现象，AI推理需要时间
- 检查网络连接是否正常
- 确认API密钥有效且账户余额充足

### 问题3：返回错误信息
**解决**：
- 查看后端控制台日志获取详细错误
- 确认API密钥是否正确
- 检查阿里云账户是否有足够的余额

## 💡 使用技巧

### 好的提问方式：
✅ "基于当前库存数据，你有什么建议？"
✅ "哪些设备逾期未还？我该怎么处理？"
✅ "分析一下最近一个月的借用趋势"
✅ "如何优化我们的库存配置？"

### 避免的提问方式：
❌ "你好"（太简单，无法提供有价值的答案）
❌ 拼写错误的设备号
❌ 与库存管理完全无关的问题

## 📊 AI助手能做什么？

1. **数据分析**：分析库存状态、借用趋势、设备分布
2. **问题诊断**：发现逾期、库存不足等问题
3. **管理建议**：提供优化建议和最佳实践
4. **智能预警**：主动提醒潜在风险
5. **决策支持**：帮助做出更好的管理决策

## 🔒 安全提示

- ⚠️ 不要将 `.env` 文件提交到Git
- ⚠️ 不要公开分享你的API密钥
- ⚠️ 定期检查API使用情况，避免费用超支
